- Lancer Docker au préalable.

- Installer les node_modules:
```code
    npm ci
```

- Lancer le serveur:
    ```code
    npm run start
    npm run start:dev
    ```

- Lancer test:
    ```code
    npm run test:e2e:postgres
    ```

Il est possible qu'il soit nécessaire de lancer les commandes en ```sudo``` dans le cas ou les droits ne seraient pas accordés.
